■ 概要<br>
EDCB (EpgDataCap_Bon) の予約状況を確認・モニタリングするためのツールです。<br>
<br>
■ 特徴<br>
本ソフトはパイプ通信とファイル監視を組み合わせたEDCB専用の管理ツールです。<br>
パイプ通信により本ソフト上から直接「予約の有効/無効」や「削除」を操作して本体へ反映できるため<br>
閲覧だけのビューアーとは異なりEpgTimer本体を開く手間を削減します。<br>
また、Reserve.txtの更新監視により予約変動を瞬時に検知して表示を更新します。<br>
定期的な1分毎の確認と更新検知（即時）を併用することでデータの正確性を保ちつつ<br>
PCへの負荷を最小限に抑えた設計となっています。<br>
<br>
■ 動作環境<br>
・Windows 10 / 11 (64bit)<br>
・必須環境: EDCB (EpgDataCap_Bon) が導入済みであること<br>
・開発環境/ランタイム: .NET 8 (自己完結/Self-containedにより同梱済)<br>
  ※本アプリは .NET 8 ランタイム(x64)を内包しているため別途インストールする必要はありません。<br>
<br>
■ 使い方<br>
1. インストール<br>
   「EDCBMonitor.exe」を適当な場所（任意のフォルダ）に配置してください。<br>
<br>
2. 初回起動と設定<br>
   「EDCBMonitor.exe」を実行してください。<br>
   初回設定後に同ディレクトリ内に設定ファイル「EM_Config.xml」が自動生成されます。<br>
   ※設定ファイルは必ずexeと同じ場所に置いてください。<br>
<br>
3. EDCBとの連携<br>
   設定画面にてご使用のEDCBの「Reserve.txt」がある場所（パス）を指定してください。<br>
   OSのファイル変更通知をフックし予約変動を瞬時に反映します。<br>
<br>
■ 自動起動について<br>
PC起動時に本ソフトを自動的に立ち上げたい場合は、Windowsのスタートアップフォルダに<br>
「EDCBMonitor.exe」のショートカットを作成・登録してください。<br>
<br>
※スタートアップフォルダの開き方：<br>
「Windowsキー + R」を押し「shell:startup」と入力してOKを押すと開きます。<br>
<br>
■ アップデート時の注意<br>
バージョン更新を行う際は念のため既存の「EM_Config.xml」を削除してから<br>
新しいバージョンのexeで再設定を行ってください。<br>
