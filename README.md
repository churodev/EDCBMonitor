■ 概要
EDCB (EpgDataCap_Bon) の予約状況を確認・モニタリングするためのツールです。

■ 特徴
本ソフトはパイプ通信とファイル監視を組み合わせたEDCB専用の管理ツールです。
パイプ通信により本ソフト上から直接「予約の有効/無効」や「削除」を操作して本体へ反映できるため
閲覧だけのビューアーとは異なりEpgTimer本体を開く手間を削減します。
また、Reserve.txtの更新監視により予約変動を瞬時に検知して表示を更新します。
定期的な1分毎の確認と更新検知（即時）を併用することでデータの正確性を保ちつつ
PCへの負荷を最小限に抑えた設計となっています。

■ 動作環境
・Windows 10 / 11 (64bit)
・必須環境: EDCB (EpgDataCap_Bon) が導入済みであること
・開発環境/ランタイム: .NET 8 (自己完結/Self-containedにより同梱済)
  ※本アプリは .NET 8 ランタイム(x64)を内包しているため別途インストールする必要はありません。

■ 使い方
1. インストール
   「EDCBMonitor.exe」を適当な場所（任意のフォルダ）に配置してください。

2. 初回起動と設定
   「EDCBMonitor.exe」を実行してください。
   初回設定後に同ディレクトリ内に設定ファイル「EM_Config.xml」が自動生成されます。
   ※設定ファイルは必ずexeと同じ場所に置いてください。

3. EDCBとの連携
   設定画面にてご使用のEDCBの「Reserve.txt」がある場所（パス）を指定してください。
   OSのファイル変更通知をフックし予約変動を瞬時に反映します。

■ 自動起動について
PC起動時に本ソフトを自動的に立ち上げたい場合は、Windowsのスタートアップフォルダに
「EDCBMonitor.exe」のショートカットを作成・登録してください。

※スタートアップフォルダの開き方：
「Windowsキー + R」を押し「shell:startup」と入力してOKを押すと開きます。

■ アップデート時の注意
バージョン更新を行う際は念のため既存の「EM_Config.xml」を削除してから
新しいバージョンのexeで再設定を行ってください。
